# Music Player Pro

A simple music player written entirely in C#. 

## Controls

|Keybind|Action|
|--|--|
|Arrow Key UP / DOWN|+/- 5% Volume|
|Arrow Key LEFT / RIGHT|+/- 5 Seconds of Playback|
|J/L|+/- 10 Seconds of Playback|
|Space|Pause|
|Ctrl + O|Open File|
|Ctrl + F|Open Folder|
